# 🔎 1. Vérification de la base de données

Assurez-vous que la base de données appdb est bien présente dans votre système de gestion de base de données.

Cette vérification permet de confirmer que l’importation initiale a réussi et que la base est accessible.

# 📋 2. Vérification des tables

Après avoir confirmé la présence de la base de données, vérifiez que toutes les tables attendues sont disponibles.

La présence des tables confirme que :

La structure de la base de données est complète

L’importation s’est déroulée sans erreur

Le schéma correspond à la configuration prévue
